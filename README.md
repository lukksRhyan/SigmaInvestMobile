## Preparativos

1. Instalar dependências

   ```bash
   npm install
   ```
3. Configurar API_KEY da brapi em services/apiClients.ts


2. Buildar aplicativo (usando o expo)

   ```bash
    npx expo start
   ```

3. Instalar o APK
